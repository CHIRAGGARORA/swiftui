## Declarative vs Imperative

Notices the change in language. In SwiftUI we update the view by changing the state (declarative) vs telling it what to do (imperative).

**Imperative**

```swift
Button {
    // show vehicles
}
```

**Declarative**

```swift
Button {
    // Change state to vehicles
}
```
